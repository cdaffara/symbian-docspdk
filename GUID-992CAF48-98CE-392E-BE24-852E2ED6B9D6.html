TDigitiserInfoV01 Class Reference

class TDigitiserInfoV01

Encapsulates information about the digitiser.

An object of this type is passed, via a TPckgBuf, to the HAL handler that deals with the HAL group function-id pair:

  • EHalGroupDigitiser, EDigitiserHalXYInfo;
