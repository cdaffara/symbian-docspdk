USB Mass Storage Server

Used to read and write data on a USB host device.

Introduction

This document describes the USB client mass storage system, which is used to interact with media through USB when connected to a host (a PC or a Symbian platform device).

USB mass storage server features

The USB mass storage system provides the following features:

  • Provides services to access the media on the device (running Symbian platform) through a USB interface for the host.

  • To use it, the drive that on the Symbian device and that is going to be available to the PC must be using the FAT file system.

  • The device running Symbian platform must be connected to a host PC.

  • The media on the device running Symbian platform has to be using the FAT file system format.
