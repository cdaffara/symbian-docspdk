MSwtPaletteData::TDirectData Struct Reference

struct MSwtPaletteData::TDirectData

Data structure for direct palettes

To extract a component from a colour: first apply the mask and then shift the result by the specified value.

To combine components in order to build a colour, for each component: first shift by the opposite of the specified value and then apply the mask. Finally, combine all three components with a OR operator.

In both cases, a positive value indicates a left shift whereas a negative one indicates a right shift.
