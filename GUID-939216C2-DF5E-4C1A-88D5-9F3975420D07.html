Rotation sensor channel

The rotation sensor channel detects the degrees of rotation of the Symbian device about each axis, in the three dimensional Cartesian coordinate system.

The following table provides details about the rotation sensor channel.

Header sensrvorientationsensor.h
Data type Continuous
Channel type id KSensrvChannelTypeIdRotationData
Data class TSensrvRotationData
Data units 0 to 359 Degrees
