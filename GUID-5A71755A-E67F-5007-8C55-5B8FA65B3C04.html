ROM File System

Describes the ROM file system.

Introduction

This is the file system is used to access code storage on execute-in place (XIP) storage e.g. NOR flash.

Background information

This is the only file system present when the device boots-up.

It is used by the composite file system to present a single drive Z: to the file server.

ROM features

The ROM filing system provides the following features:

  • The only file system that is built into the file server and hence is the only one present when the device boots up.

  • Used to access code storage on execute in place (XIP) media.

ROM filing system limitations

The following are known limitations for the ROM filing system:

  • Write operations cannot be carried out with this filing system.
