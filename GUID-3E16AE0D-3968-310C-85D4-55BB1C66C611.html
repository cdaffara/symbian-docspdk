RSat::TSendDtmfV5 Class Reference

class RSat::TSendDtmfV5 : public RSat::TSendDtmfV1

Contains a SEND DTMF proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifySendDtmfPCmd() for more information about the command.

Request notification for SEND DTMF with RSat::NotifySendDtmfPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TSendDtmf" class. For Symbian OS versions up to and including v8.1a and 8.1b use RSat::TSendDtmfV1Pckg, a packaged RSat::TSendDtmfV1. For Symbian OS 9.2, use RSat::TSendDtmfV5Pckg. For later versions of Symbian OS, check for an updated API (RSat::TSendDtmfV6 etc).
