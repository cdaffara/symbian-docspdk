Porting Tutorial

This section describes changes and the instructions required to migrate from apps to exes, control panel plug-ins to Symbian OS v9.1 and later, exe-apps to a data caged directory structure, data recognizers to the secure version of the Symbian platform and notifiers to Symbian OS v9.1.

  • Porting Applications to EXEs
  • Porting Control Panel Plug-ins
    The purpose of this document is to help migrate polymorphic DLLs to plug-ins as applications.
  • Porting Data-caged Applications
    This document describes the changes required to migrate exe-apps to a data caged directory structure.
  • Porting Data Recognizers to Secure Platform
    This page describes how to migrate data recognizers to the secure version of the Symbian platform.
  • Porting Notifiers to Secure Platform
    This page describes how to migrate notifiers to Symbian OS v9.1.
