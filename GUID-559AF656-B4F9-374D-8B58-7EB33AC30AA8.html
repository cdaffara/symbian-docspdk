RMobileLocationServices::TMeasurementControlV7 Class Reference

class RMobileLocationServices::TMeasurementControlV7 : public RMobilePhone::TMultimodeType

This class contains the GPS assistance data needed by the UE to compute the location of the UE. The UE computes the location once it receives all required information.
