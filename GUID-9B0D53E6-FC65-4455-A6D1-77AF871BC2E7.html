WMDRM

Windows Media DRM (WMDRM ) 10 is a Microsoft technology for protecting and delivering media content. It is available on a computer, portable device, or network device.

Some devices based on the Symbian platform support WMDRM in addition to OMA DRM. However, DRM Audio Player API does not support WMDRM, and therefore the devices can only play the content offline.

For more information on WMDRM, see the Microsoft Web site.

Note:

When developing your application, keep in mind that OMA DRM and WMDRM are not interoperable.

  • WMDRM Engine Overview
    WMDRM Engine is a component of WMDRM collection. It provides interfaces to perform WMDRM implementations, such as managing WMDRM licenses and decrypting WMDRM content. Applications that require WMDRM functionality must interact with WMDRM plug-ins and Camese components by implementing the interfaces provided by WMDRM engine.
