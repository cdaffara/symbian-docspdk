RSat::TReceiveDataV2 Class Reference

class RSat::TReceiveDataV2 : public RSat::TSatPCmdBase

Contains a RECEIVE DATA proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyReceiveDataPCmd() for more information about the command.

Request notification for RECEIVE DATA with RSat::NotifyReceiveDataPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TReceiveData" class. For Symbian OS versions up to and including v8.1a and 8.1b use RSat::TReceiveDataV1Pckg, a packaged RSat::TReceiveDataV1. For later versions of Symbian OS, check for an updated API (RSat::TReceiveDataV2 etc).
