Plug-in Framework Concepts

The Plug-in framework has client-server architecture. Clients of the plug-in framework use the C++ interface provided by that framework. The interface definition promises services to the client and the interface implementation provides the services promised by the interface. The Platform Security Architecture ensures security in using the plug-in framework.

  • ECom Architecture
  • Interface definition
  • Interface Implementation
  • Interface Extensibility
    Symbian^3 onwards, the plug-in framework supports interface extensibility. An extended interface extends the existing functionality of the original interface (instantiation interface). By extending interfaces, the framework developers need not change the existing interface specifications to support new requirements.
