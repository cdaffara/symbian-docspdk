Telephony API for Applications

The telephony API gives applications access to Telephony services.

The telephony API for applications provides a simple interface for the applications developers to access the telephony functionality. This API is a subset of the multimode telephony API that provides a rich set of telephony functionalities to the handset manufacturers.
