Telephony Server Plug-ins (TSY)

A TSY is an interface between Symbian platform and the modem. This topic describes the telephony server plug-ins and the main function they provide to the licensees.

A new TSY is needed when new hardware is introduced. Symbian platform provides a layered approach for easy TSY implementation. These layers are:

  • Common TSY or CTSY: This is a generic layer that is independent of the baseband

  • Licensee TSY or LTSY: This is a thin layer that interfaces to the baseband software

The LTSY implements the functionality provided by the CTSY. The LTSY is the layer that a licensee needs to adapt when a new baseband is introduced.
