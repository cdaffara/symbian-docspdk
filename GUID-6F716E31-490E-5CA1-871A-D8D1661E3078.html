Feature Manager Installation Tutorials

This document introduces tutorials on the use of Feature Manager to enable and disable features on a phone using the feature installer.

A phone is typically built with many features which require to be disabled or enabled for the first time. These tutorials tell you how to do so using the feature installer.

  • Installing Features with Feature Manager
    Provides instructions for adding features when installing a new software component.
  • Uninstalling Features with Feature Manager
    Provides instructions for removing features when uninstalling a software component.
