CListenerDopplerMessageHandler Class Reference

class CListenerDopplerMessageHandler : public CMMFObject

This class encapsulates the message handing function of the doppler effect. All custom commands send by the Doppler Proxy are routed to this class. The custom commands are dispatched to the appropriate handler.
