const TUint KSoIpsecEndpoint

const TUintKSoIpsecEndpoint

Get/Set the End Point address.

The option data is TNameRecord. This definition allows easy use of RHostResolver results as a parameter to IPSEC end point.

  • iFlags are ignored

  • iName is the name of the end point

  • iAddr is the address contained in the end point (port is ignored).
