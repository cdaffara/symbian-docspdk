How to reset

The following code resets the document. Resetting deletes all text and formatting, leaving the end of text paragraph delimiter. Because resetting deletes the documentâ€™s content and formatting, a call to HandleGlobalChangeL() is required to update the formatting and the view.

// Reset document. 
iRichText->ResetL(); // change whole document
iTextView->HandleGlobalChangeL();
