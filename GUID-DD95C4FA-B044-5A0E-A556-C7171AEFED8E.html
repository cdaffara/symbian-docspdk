Streaming Session Sequence

This section explains the set-up and tear-down of streaming session on the Symbian platform, using sequence diagrams.

Note:

  • This illustration assumes a separation of application and streaming player middleware (multimedia processes). The separation is optional. However, if it is not done, the application process will include the components displayed in the multimedia process and will be forced to possess trust status and Digital Rights Management (DRM) capability.

  • The process boundary between the Multimedia process and the CAF streaming agent server process is not enforced by the Symbian platform. However, it is strongly recommended, otherwise no DRM trusted path can be established for protected content.
