Reference

The Reference provides a detailed description of programming interfaces and error codes.

This section contains the Nokia Symbian^3 C++ API Reference which provides documentation for the public and platform classes and header files. It is organized according to the Symbian System Model.

The reference also contains information for patchable constants, which are DLL constants that can be set when building a ROM image, and system panics, which are the errors codes used when the system terminates a program with a serious fault.
