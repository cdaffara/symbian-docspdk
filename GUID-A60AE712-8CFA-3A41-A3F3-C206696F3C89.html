ImageProcessor::MImgProcessorPluginPanoramaInput Class Reference

class ImageProcessor::MImgProcessorPluginPanoramaInput

Mixin class for implementation by providers of extended functionality for Image Processor Plugin. The class provides a function to set an panoramic image as input to Image Processor. The implementation of the class is provided through MImgProcessorPlugin::Extension(TUid aExtension). CImgProcessor MImgProcessorPlugin::Extension(TUid aExtension)
