Telephony Server

This section describes the concepts and APIs of the Telephony Server. The telephony API enable the developers to implement the telephony related functions in Symbian platform.

  • Core Telephony
    The Core Telephony API provides a generic interface to start, control and terminate telephone calls. The core telephony functions are provided in terms of phone, line and call.
  • Multimode Telephony
    The ETel multimode API enables access to common mobile telephony services for a number of air-interface cellular standards or modes. It also enables access to specialised services within the GSM and CDMA IS-95 standards and provides an API that allows switching between supported modes. Currently, GSM, CDMA (IS-95 and cdma2000) and WCDMA (3GPP rel '99) modes are the fully supported modes.
  • SIM Application Toolkit (SAT) API
    This section provides information about the SIM Application Toolkit (SAT) API interface in Symbian platform.
  • Packet Data Telephony
    This section provides the information related to the packet telephony API.
