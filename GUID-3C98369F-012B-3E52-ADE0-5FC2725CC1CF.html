TInt HandleGetCurrentActivePinReqL()[pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the EMmTsySecurityGetSimActivePinStateIPC request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackSecurityGetCurrentActivePinComp()

Implementation of this interface should prepare the updating the information of the currently active PIN.
