Configuring LBS to use the Privacy Notifier

To use the Privacy Query and Notification API to create a notifier a licensee must thirdly configure LBS to use the Privacy Notifier.

The LBS Administration setting KLbsSettingPrivacyHandler must be set to CLbsAdmin::EPrivacyHandleByAdvancedNotifier to configure LBS to use the Privacy Q&N Notifier. See LBS Administration for details of how to configure this setting either through the LBS Administration API or in the LBS central repository initialisation file.
