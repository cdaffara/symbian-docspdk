Secure Software Install Device Tools Tutorials

This section lists the tasks that can be performed using the Secure Software Install Device Tools.

  • Creating Swicertstore.dat File
    During installation of a native software SWI checks the certificates in the swicertstore.dat file against the certificates with which the SIS files are signed to assure if the native software being installed is from a trusted source.
  • Using Console-based SWI
    Console-based SWI tool installs or uninstalls native software using an EShell interface. Note: This tutorial is useful for S60 emulator only.
