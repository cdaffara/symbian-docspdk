RSat::TPerformCardApduRspV2 Class Reference

class RSat::TPerformCardApduRspV2 : public RSat::TSatPCmdBase

Contains your response to a PERFORM CARD APDU proactive command. Pass this to the UICC with RSat::TerminalRsp(). See ETSI TS 102 223 and RSat::NotifyPerformCardApduPCmd() for more information about the command.

For Symbian OS versions from 7.0s to v8.1a and v8.1b pass RSat::TerminalRsp() a RSat::TPerformCardApduV2Pckg (a packaged TPerformCardApduRspV2). For later versions of Symbian OS, check for an updated API (RSat::TPerformCardApduRspV3 etc).

Member Data Documentation

TAdditionalInfo iAdditionalInfo

TAdditionalInfo iAdditionalInfo

This should be the card's response to the APDU, is there is one. Otherwise, some error values of iGeneralResult require additional information, which goes here.

TPCmdResult iGeneralResult

TPCmdResult iGeneralResult

General result of the proactive command. Set this as described in the proactive commands versus terminal response table in ETSI TS 102 223.

TUint iInfoType

TUint iInfoType

If there a response to the APDU then set this to RSat::KRApdu. If something goes wrong, then some error values of iGeneralResult require additional information, in which case set this to RSat::KMeProblem. Otherwise set to RSat::KNoAdditionalInfo.
