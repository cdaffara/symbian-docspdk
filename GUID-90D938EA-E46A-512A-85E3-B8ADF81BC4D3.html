Bluetooth GPS PSY Tutorials

These tutorials describe how to configure and use the Bluetooth GPS PSY.

  • Modifying the Bluetooth GPS PSY Settings
    This topic describes how to modify the settings of the Bluetooth GPS PSY.
  • Including the Bluetooth GPS PSY in ROM
    This describes how to include the PSY in ROM using buildrom macros.
  • Packaging the Bluetooth GPS PSY in a SIS File
    This describes how to package the Bluetooth PSY so it can be installed by a mobile device user. It is only necessary to install the PSY as a SIS file if it is not already installed in ROM by a device creator.
  • Opening a Session with the Bluetooth GPS PSY
    This topic describes how to open a session with the Bluetooth GPS PSY.
