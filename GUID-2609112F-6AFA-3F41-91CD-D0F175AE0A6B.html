RSat::TLocalInfoV6 Class Reference

class RSat::TLocalInfoV6 : public RSat::TLocalInfoV3

Contains a PROVIDE LOCAL INFORMATION proactive command issued by the UICC. See 3GPP TS 31.111 Release 6, v 6.8.0, 2006-03 and RSat::NotifyLocalInfoPCmd() for more information about the command.

Request notification for PROVIDE LOCAL INFORMATION with RSat::NotifyLocalInfoPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TLocalInfo" class. RSat::TLocalInfoV6Pckg RSat::NotifyLocalInfoPCmd()
