RMobilePhone::TGbaBootstrapAuthDataV8 Class Reference

class RMobilePhone::TGbaBootstrapAuthDataV8 : public RMobilePhone::TImsAuthenticateDataV5

The class is identical to its base class except for its GBA Bootstrapping specific V8 extension id. Contains parameters to hold the input as well as the output parameters for the GBA authentication in the Bootstrapping mode. The parameters are used as follows:

Input: RAND,AUTN Output: RES (if the authentication procedure is successful) or AUTS (contains the failure response in case of unsuccessful authentication procedure) See section 7.1.1.4 and 7.1.2.3 of 3GPP TS 31.102 v6.18.0
