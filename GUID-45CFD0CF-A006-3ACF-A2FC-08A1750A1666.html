TUPnPSelectDefaultResource Class Reference

class TUPnPSelectDefaultResource : public MUPnPResourceSelector

A helper selector that will select the original or most likely resource from an item, using UPnPItemUtility::ResourceFromItem. Most clients will find this practical. This class could be thought as a DEFAULT implementation for MUPnPResourceSelector.
