FAT File System

FAT File System is used for storing user data on NAND Flash, internal RAM drives and removable media.

  • FAT File System Guide
    Describes the FAT (File Allocation Table) file system that was originally devised for MS-DOS.
  • Leaf Directory Name Cache Configuration Tutorial
    Describes how to configure the cache for Leaf Directories in the FAT file system.
  • Directory Cache Tutorial
    Describes how to configure the directory cache in a FAT file system enabled drive.
