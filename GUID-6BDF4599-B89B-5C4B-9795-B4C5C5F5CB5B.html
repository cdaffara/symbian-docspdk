SMS Stack Configuration

The SMS stack derives settings from the Comms database. The settings in the the SMS protocol ESK file can also be configured.

The standard Symbian platform implementation has one SMS stack. The SMS Stack derives some settings from the Comms Database. For more information about the Comms Database, see the Comms Database Reference. The SMS stack also has some settings available in the SMS protocol ESK file. For more information about the ESK file settings used by SMS, see the the SMS Stack Architecture.
