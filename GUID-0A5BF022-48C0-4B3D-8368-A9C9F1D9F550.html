Stylus scrolling pop-up

Stylus scrolling pop-up component is used only to display scrolling indicators, for example, page “1/3” of the editor or viewer.

The component used in stylus scrolling pop-up is information pop-up. The exact content of the stylus scrolling pop-up is specified in applications.

The stylus scrolling pop-up can be used only when the scrollbar is used with stylus. The stylus scrolling pop-up appears to the display when the user is scrolling the content and a page break can occur to the scrolling. This means that stylus pop-up is required only when there is more than one page to scroll. The stylus scrolling pop-up disappears when the user releases the scrollbar (stylus up event on the scrollbar).
