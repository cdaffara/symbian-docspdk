Application Architecture Concepts

Application Architecture Concepts section provides a detailed description on Server applications and MIME Content policy. Server applications basically provide services and contains a server to which other applications can connect. APPARC provides an API known as MIME Content policy API to check MIME types and file extensions against Closed Content list.
