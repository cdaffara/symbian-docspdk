Types of Sensor Channels

  • Accelerometer Sensor Channel
    The accelerometer sensor channel measures the acceleration and detects movement gestures of the Symbian device.
  • Rotation sensor channel
    The rotation sensor channel detects the degrees of rotation of the Symbian device about each axis, in the three dimensional Cartesian coordinate system.
  • Orientation sensor channel
    The orientation sensor channel indicates the orientation of the Symbian device, for example, display up or display down.
  • Accelerometer Double-Tap Sensor Channel
    The accelerometer sensor channel detects a double-tap on the device where the taps occur in quick succession and in the same direction.
  • Proximity Monitoring Sensor Channel
    The proximity monitoring sensor channel indicates how close the Symbian device is to, for example, the user's hand or ear.
  • Ambient Light Sensor Channel
    The ambient light sensor channel indicates the current brightness level.
  • Magnetic North Sensor Channel
    The magnetic north sensor channel indicates the difference in angle between the Symbian device and magnetic north, in degrees.
  • Magnetometer XYZ Sensor Channel
    The magnetometer XYZ sensor channel indicates the strength of the geomagnetic flux density in the X, Y and Z axes.
