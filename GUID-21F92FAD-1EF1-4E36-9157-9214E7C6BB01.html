Host name resolution

Sometimes it is necessary to authenticate the remote device that is connected through a public IP network. You can do this by querying and comparing the domain name system (DNS) name of the device to its IP address. To perform host resolve queries to both directions, use the RHostResolver class.

Note that this is not a secure way to authenticate remote devices, because some clients (for example, in a local network) may not have DNS names at all . For reliable authentication and secure communication, use stronger methods like key exchange or secure protocol solutions.
