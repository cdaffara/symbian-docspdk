const TInt KFdcInterfaceV1

const TIntKFdcInterfaceV1

The UID of this FDC interface. If the FDC API ever has to change, a new UID and associated M class will be created. New FDC implementations may implement the new API. Old (non-updated) FDCs will still work because the FDF examines at run-time which interfaces are supported by any particular FDC.
