RSat::TLaunchBrowserV5 Class Reference

class RSat::TLaunchBrowserV5 : public RSat::TLaunchBrowserV2

Contains a LAUNCH BROWSER proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyLaunchBrowserPCmd() for more information about the command. Symbian OS support for this command is available from v7.0s onwards.

Request notification for LAUNCH BROWSER with RSat::NotifyLaunchBrowserPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TLaunchBrowser" class. For Symbian OS 7.0s, 8.0 and 8.1 use RSat::TLaunchBrowserV2Pckg, a packaged RSat::TLaunchBrowserV2. For later versions of Symbian OS, check for an updated API (RSat::TLaunchBrowserV6 etc).
