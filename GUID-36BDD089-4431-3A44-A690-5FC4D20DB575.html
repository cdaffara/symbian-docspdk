RMobileLocationServices::TEllipsoidPointAltitudeV1 Class Reference

class RMobileLocationServices::TEllipsoidPointAltitudeV1

The description of an ellipsoid point with altitude is that of a point at a specified distance above or below a point on the earth's surface. This is defined by an ellipsoid point with the given longitude and latitude and the altitude above or below the ellipsoid point. 3GPP TS 23.032
