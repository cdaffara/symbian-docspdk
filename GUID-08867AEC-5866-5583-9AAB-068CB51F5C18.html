Platform-Specific Source Code

The platform-specific layer of the bootstrap must implement a set of standard functions and definitions that are used by the generic layer.

  • Public Functions
    Describes how to implement the three exported functions of the platform-specific layer.
  • Boot Table
    Describes the boot table.
  • Boot Table Functions
    Describes how to implement the functions that the bootstrap implementation must provide.
  • Boot Table MMU Permission and Cache Attribute Definitions
    Lists MMU attributes that the bootstrap implementation must provide.
