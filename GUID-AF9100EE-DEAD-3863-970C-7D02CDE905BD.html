MWsGraphicsContext::TTextParameters Class Reference

class MWsGraphicsContext::TTextParameters

Parameters used in drawing text within supplied context. It is used by MWsGraphicsContext::DrawText() and MWsGraphicsContext::DrawTextVertical() family of API's to draw text from iStart to iEnd withing the supplied text descriptor.
