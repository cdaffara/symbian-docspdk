Using Alarm Server

This tutorial explains how clients can use the services provided by Alarm Server.

Introduction

Using the RASCliSession class of Alarm Server, clients can add, delete and update alarms, set alarm status, and so on. The procedure section lists and explains how to use the services provided by Alarm Server.
