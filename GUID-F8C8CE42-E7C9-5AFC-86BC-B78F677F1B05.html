Device Services Guide

Device Services provides device specific services to control phone resources and establish remote controlled connection with the external devices. It also monitors the phone resources and provides stage shutdown mechanism.

This guide provides detailed information on the Device Services subsystem.

  • Accessory Services
  • Resource Management
    Resource Management collection provides a number of components for high level access to phone resources, including power, light and vibration.
  • System Health Management
    System Health Management contains the System Health Manager component, formerly known as the System Monitor, which monitors and restarts processes and applications.
  • System State Management Architecture
    The startup of the Symbian platform is modularised, so that different classes of servers or applications can be launched by different components. The System State Manager (SSM) extends the existing Generic Start-up Architecure (GSA) to include system state management and shutdown. The expanded GSA is called the System State Management Architecture (SSMA). The SSMA handles system state manager and shutdown and system state plug-in.
  • Sensor Services
    The Sensors Services collection provides Sensors Framework, which allows clients to request and receive sensor information from the different sensors on a device.
