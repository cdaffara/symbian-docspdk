Magnetometer XYZ Sensor Channel

The magnetometer XYZ sensor channel indicates the strength of the geomagnetic flux density in the X, Y and Z axes.

The following table provides information about the magnetometer XYZ sensor channel:

Header sensrvmagnetometersensor.h
Data type Continuous
Channel type id KSensrvChannelTypeIdMagnetometerXYZAxisData
Data class TSensrvMagnetometerAxisData
Data units Tesla (scaled)
Note: Each packet of TSensrvMagnetometerAxisData data contains two sets of data, raw and calibrated. Auto calibration must be enabled for the calibrated data fields to contain valid data.
