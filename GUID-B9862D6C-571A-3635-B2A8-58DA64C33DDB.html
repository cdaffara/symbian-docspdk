UnattendedTransfer(const TDesC &)

TInt UnattendedTransfer(const TDesC &aTransferTarget)[pure virtual]
Unattended transfer to given address. Call is requested to be transferred to given address. Afterwards call will be disconnected from CCE side. If transfer fails a new incoming call will be received to reconnect to recipient. Reconnected call is initiated from recipient end. Actual result like status, events and errors come via MCCECallObserver class. This is not used for CS.
Since
S60 3.2
Pre-condition
Call state is MCCECallObserver::ECCPStateConnected or MCCECallObserver::ECCEStateHold

Parameters

const TDesC & aTransferTargetAddress of the target
