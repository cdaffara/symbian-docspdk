const TUint32 KCbsServerStarted

const TUint32KCbsServerStarted

This key is used to indicate the status of CBS Server to other applications. The value is set to '1' when server is started, otherwise it is '0'.

Possible values: 0: Default value, CBS Server is not running 1: Indicates that CBS Server is running
