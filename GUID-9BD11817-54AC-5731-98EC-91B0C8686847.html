What Are Envelope Commands

This section describes the envelope commands and the functions they provide to the client applications.

The envelope commands request the TSY to package the messages related to the SAT. The envelope commands enable the communication between the SAT application and SAT engine. The envelope commands cannot be used to instruct the device. This section provides some envelope commands supported by Symbian platform.

Envelope command functions

Envelope commands allow the SAT engines to:

  • get the SAT features that the device supports.

  • send the menu item selected by the user to the client applications. The menu is created by a proactive command.

  • notify the SAT engine of any changes made to the earlier request by a call or a PDP context.

  • notify the SAT engine of different events in the device like idle screen, channel, connection and browser termination.

  • notify the client applications of new Multimedia Messages (MMS).

  • get the unstructured service data from the network.
