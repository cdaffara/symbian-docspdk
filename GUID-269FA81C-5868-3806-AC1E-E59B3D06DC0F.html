TESMRAlarmInfo Class Reference

class TESMRAlarmInfo

TESMRAlarmInfo defines ES MR alarm information object.

Alarm information objects are defined in resource files. Resource structure ESMR_ALARM_INFO defines alarm info object in resource files.
