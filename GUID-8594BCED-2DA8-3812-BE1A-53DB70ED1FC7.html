TTaskConfig2 Class Reference

class TTaskConfig2 : public TTaskConfig

Extended version of TTaskConfig used by the SetConfig() method to configure the sample rate, stereo mode and number of channels of a CMdfHwDevice plugin. The configuration of HwDevices is device specific.
