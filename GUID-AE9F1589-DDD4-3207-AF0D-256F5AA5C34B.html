MSwtImage Class Reference

class MSwtImage : public MSwtBitmapDrawable

Interface class for the org.eclipse.swt.graphics.Image class.

Images are not intended to be disposed of while they are referenced by some control and/or GC. Nonetheless, a broken application may loosely follow the SWT coding guidelines and fail to do so, thereby causing an access violation.

In order to prevent such behaviour, images are reference counted, see MSwtRefCounted.

Retrieves a const reference to a scaled copy of the native bitmap. Allocates memory in the object even if the object is const (see mutable). The allocated copy is owned by the image object.
