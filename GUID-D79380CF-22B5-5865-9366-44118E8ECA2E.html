Open Environment Core (P.I.P.S.)

P.I.P.S. I s P OSIX on S ymbian Platform (P.I.P.S.). It provides an API layer, above the Symbian platform native APIs, that is more closely aligned with industry standard APIs making Symbian software development more accessible to Symbian developers who program using the C language.
