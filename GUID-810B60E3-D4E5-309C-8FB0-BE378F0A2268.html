RSat::TPowerOffCardV2 Class Reference

class RSat::TPowerOffCardV2 : public RSat::TSatPCmdBase

Contains a POWER OFF CARD proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyPowerOffCardPCmd() for more information about the command.

Request notification for POWER OFF CARD with RSat::NotifyPerformCardApduPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TPowerOffCard" class. For Symbian OS versions from 7.0s to v8.1a and 8.1b use RSat::TPowerOffCardV2Pckg, a packaged RSat::TPowerOffCardV2. For later versions of Symbian OS, check for an updated API (RSat::TPowerOffCardV3 etc).
