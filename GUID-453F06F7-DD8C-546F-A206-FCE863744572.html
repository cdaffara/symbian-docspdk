Extended Light API Tutorial

This tutorial describes how to use the Extended Light API of HWRM.

Introduction

Extended Light API is similar to the Light API but it supports more target devices. It can be used with or without a notification. When the notification is required, a callback pointer is implemented in the instance of the CHWRMExtendedLight. The callback pointer is provided by the MHWRMExtendedLightObserver class.

Using Extended Light API, you can perform any of the tasks listed in the procedure section:

Procedure

  • Creating an instance

  • Retrieving the supported targets

  • Retrieving the status of the lights

  • Reserving the lights

  • Switching ON the lights

  • Switching OFF the lights

  • Blinking the lights

  • Releasing the reserved lights

  • Releasing the instance of Extended Light API
