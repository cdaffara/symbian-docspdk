Concepts

This section provides a summary of concepts related to the multimode telephony. The multimode telephony APIs provide rich mode specific telephony functionalities.

  • Multimode Phone
    RMobilePhone encapsulates the properties of the phone that do not relate to a particular line or call. It is derived from theRPhone interface of core telephony.
  • Multimode Line
  • Multimode Call
    RMobileCall encapsulates information and functionality for a particular call. It is derived from the RCall interface of core telephony
  • Multimode Messaging
    The Multimode telephony API provides a number of specialised messaging classes that are derived from RTelSubSessionBase. One of the classes is responsible for SMS, another for Broadcast and another for USSD messaging.
  • Multimode Stores
  • GBA_U and MBMS Authentication
  • Mailbox Dialling Numbers
  • Message Waiting Indicators
  • Wireless LAN Parameters
