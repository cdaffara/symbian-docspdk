Forms without the options softkey

In case a form does not need any context-specific functions in the Options menu, the softkey interface can be the same as in queries: the left softkey is Done and the right softkey is Cancel. Done accepts the contents and returns, whereas the Cancel function discards all changes in the form and returns.
