OMA DRM v2.1

The following OMA DRM v2.1 features are supported:

  • Metering

    Allows the rights issuer to collect content usage data. Mostly used in subscription-based services.

  • O4a and O4v media type extensions

    O4a indicates that the media type of the content object in the DRM content format is audio. O4v indicates video content.

  • WAP binary XML (WBXML) rights object acquisition protocol (ROAP) trigger

Use the DRM CAF Agent API to implement the OMA DRM v2.1 features in applications.
