Messaging Application Base

The Messaging Application Base is a collection of messaging-related services.

The following are the components of the Messaging Application Base collection:

  • OBEX MTM
    This section describes concepts and functions related to OBEX MTM. The client applications can use the APIs that are provided by this MTM to send one or more objects (attachments) using IR or Bluetooth.
  • SMIL Parser
