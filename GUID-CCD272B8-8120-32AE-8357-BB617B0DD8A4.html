MeshMachine::CNodeParallelActivityBase Class Reference

class MeshMachine::CNodeParallelActivityBase : public MeshMachine::CNodeActivityBase

Activity type that generates a unique activity id every time it is instantiated. The id is generated from the <TNodeActivityId::EActivityParallelRangeMin,TCFNodeActivityId::EActivityParallelRangeMax> range. The id should never be found in a static activity map. A new instance id is created rather than adding a new originator to the existing activity.
