PSY Tester Tutorial Overview

This document explains how to use the PSY Tester.

Readers should be familiar with PSY concepts and the purpose of the PSY Tester.

PSY Tester is used by device creators to test developed PSYs. PSY Tester can be used on an emulator or on a phone.
