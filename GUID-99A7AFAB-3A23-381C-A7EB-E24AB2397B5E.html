TBtreeMark Class Reference

class TBtreeMark

An iterator for a B-tree.

Functions that use a TBtreeMark are faster than those that use a TBtreePos, and can be used with a broken B-tree.

The class has no public members. Clients use it by getting TBtreeMark objects from some TBtree functions, and passing them into others.
