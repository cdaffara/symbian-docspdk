RSqlBufFlat::TBufFlat Struct Reference

struct RSqlBufFlat::TBufFlat

TBufFlat structure represents the "SysData" part of the flat buffer (see RSqlBufFlat comments), beyond which begins the flat buffer header and "Data" part.

TBufFlat structure contains information about:

  • Flat buffer elements count;

  • Flat buffer header size;

  • The size of used part of the flat buffer;
