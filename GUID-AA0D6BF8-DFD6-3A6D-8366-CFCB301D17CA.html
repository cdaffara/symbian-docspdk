Allocates and constructs a CRequestor object. The function returns NULL if there is insufficient memory. Slightly unusual construction is to ensure that iData can never be NULL even if someone inherits from this class.

Parameters

TRequestorType aTypeidentifies the type of requestor, a service or a contact.
TRequestorFormat aFormatdetermines the type of data held in aData
const TDesC & aDatais requestor data. Can be a telephone number, a URL etc.
