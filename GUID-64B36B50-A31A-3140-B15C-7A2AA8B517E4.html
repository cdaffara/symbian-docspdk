ImageProcessor::Plugin::MEffectMoonlight Class Reference

class ImageProcessor::Plugin::MEffectMoonlight : public ImageProcessor::Plugin::MEffect

Plugin API for the moonlight photoart effect. The TEffectMoonlight uses this interface class to get access to the image prosessor plugin effect. Once this has been constructed, calls to method functions of TEffectMoonlight are passed verbatim to this interface. For further description of required functionality, see TEffectMoonlight.
