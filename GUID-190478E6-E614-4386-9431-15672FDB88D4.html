About Symbian^3

This section provides an overview of Symbian^3 and its new features.

  • Introduction to Symbian^3
    The Symbian platform, is a comprehensive open-source software platform for Symbian-based devices. The platform is used by device manufactures to build devices, and by third-party developers to create after-market applications for installation onto devices.
  • What's new in Symbian^3
    Symbian^3 introduces a lot of new features that enhance communication, connectivity, multimedia, graphics, security and the overall usability of Symbian devices. These features benefit operators, application developers and device creators by making it easier to create feature-rich devices and applications.
