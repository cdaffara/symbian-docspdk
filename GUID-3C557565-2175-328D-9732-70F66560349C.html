CBassBoostMessageHandler Class Reference

class CBassBoostMessageHandler : public CMMFObject

This class encapsulates the message handing function of the bassboost effect. All custom commands send by the BassBoost Proxy are routed to this class. The custom commands are dispatched to the appropriate handler.
