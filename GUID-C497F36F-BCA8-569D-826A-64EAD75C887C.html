Locale Settings

Provides access to locale-sensitive information.

  • Locale Settings Overview
    Provides a number of APIs for configuring standard location specific data.
  • Introduction to locale settings
    Applications use TLocale as their reference for the conventions in use on a particular device.
  • Date and time formatting
    Describes how to format date and time.
  • Units of measurement
    In some locales, long and short distances may be expressed using different systems of measurement.
  • How to set the locale
    Use TLocale::Set() to set the system-wide locale settings.
  • How to format the date and time
    Format the date with locale-dependent settings, including the date separator characters, and format date as European, American or Japanese
  • How to format date and time independent of locale
    Explains how to format and display universal time, current time, date independent of locale settings.
  • How to format date as text
    Formats the day number, date suffix and month name as text into a supplied descriptor.
  • How to implement universal time and home time
    Add universal time offset to display the home time.
  • How to format currency values
    Describes how to format currency values for current locale.
