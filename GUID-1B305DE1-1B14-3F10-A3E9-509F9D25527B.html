CAknsAnimationValue Class Reference

class CAknsAnimationValue : public CBase

Animation value data class. Animation value class encapsulates the information of a single animation value (animation value uid, parameters for animation value and timing model reference id) that is used with animation command item data objects.

The class is not intended for derivation outside the library.
