voidHandleServerAppExit(TIntaReason)[pure virtual]

Receives server exit notification. Implementers of this interface must override this function to receive notification of server app exit reasons.

Parameters

TInt aReasonThe reason that the server app exited, this may be a command ID from the UI if exit was triggered from the UI, or an error code if the server app exited unexpectedly.
