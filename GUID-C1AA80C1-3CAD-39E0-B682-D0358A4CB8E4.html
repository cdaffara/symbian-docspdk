Error(TInt)

voidError(TIntaErr)[pure virtual]

Any fatal error which occurs during the send/receive session is reported to the client by a call to this function. Fatal errors are errors which can t be recovered from (e.g.: connection down) in which case MBca has become defunct and can no longer be used to send and receive data

Parameters

TInt aErrTThe fatal error code from the session failures propagated to the upper layer of the communications stack . Non fatal errors keep the MBca2 functional which in the context of MBca2 implementation may include KErrNoMemory, KErrCommsParity, KErrCommsOverrun, KErrCommsFrame.
