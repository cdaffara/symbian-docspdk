const TInt KRemConBearerInterface1

const TIntKRemConBearerInterface1

The UID of the bearer API. If the bearer API ever has to change, a new UID and associated M class will be created. New implementations of CRemConBearerPlugin may implement the new API. Old (non-updated) bearers will still work as long as RemCon supports the old API.
