Category: C32-Fault

Panic due to internal error of Serial Comms Server (or "Comms Server").

0

Failed to install active scheduler or error running the scheduler.

1

Failed to create server.

2

Failed to start server.

3

Could not allocate a Clean Up Stack.

4

API or function call not implemented yet.

5

Reference counting error.

6

NULL pointer passed to API.

7

Unable to allocate global timer, commonly caused by a memory allocation failure.
