Location Request Management

The Location Request Management collection contains components that handle requests for location request information both from installed applications and from the network.

  • Location Request Management Overview
    This document describes the Location Request Management collection.
  • Location Server
    The Location Server handles location requests from installed applications.
  • Network Request Handler
    The Network Request Handler processes privacy and location requests received from the network.
