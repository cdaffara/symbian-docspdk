Windowing Collection

The Windowing collection contains two components—the Window Server, which manages the use of the screen by applications, and the Window Server Plugins component.

Figure 1. The Windowing collection
  • Windowing Example Code
    Example code for the Windowing collection.
  • Window Server Component
    The Window Server component manages the use of the screen by applications and handles key and pointer events.
  • Window Server Plugins Component
    The Window Server Plugins component consists of the ScreenPlay render stage plug-ins.
