HandleGetPacketAttachModeReqL()

TInt HandleGetPacketAttachModeReqL()[pure virtual]

The CTSY Dispatcher shall invoke this function on receiving the EPacketGetAttachMode request from the CTSY.

It is a request call that is completed by invoking CCtsyDispatcherCallback::CallbackPacketServicesGetPacketAttachModeComp()

Implementation of this interface will return the attach mode. However if the call is being made during a packet service open process it will skip the get attach mode call.
