E32EpocExpSymInfoHdr Class Reference

class E32EpocExpSymInfoHdr

Structure contained in the export directory in text section of the stdexe/stddll. It contains information on the names of symbols exported by this stdexe/stddll and pointers to a E32EpocExpSymInfoHdr structure of any stddlls that are dependencies of this stdexe/stddll.

This is not used for emulator images see E32EmulExpSymInfoHdr below. E32EmulExpSymInfoHdr
