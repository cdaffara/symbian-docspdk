SMS Stack for GSM Networks

This section provides the summary of concepts and tutorials related to the SMS stack for the GSM networks. The client applications can access the SMS stack using socket server APIs.

The SMS stack provides some low level services that enable the client applications to send and receive the SMS messages.

  • Concepts
    This section lists the concepts and architecture of the SMS Stack.
  • SMS Stack Tutorial
    This tutorial describes how to send and receive messages using the SMS stack.
  • Alternative Encoders Tutorial
    This tutorial describes how to use the alternative character encoders.
  • SMS Stack Configuration
    This section describes how to configure the SMS stack using ESK files. ESK is a socket server (ESOCK) configuration file.
  • SMS Stack Configuration for Class 0 Messages
    This section describes how Class 0 SMS messages are handled in Symbian platform.
  • Reference
    This topic provides a summary of related documentation for the SMS stack to which you can refer.
