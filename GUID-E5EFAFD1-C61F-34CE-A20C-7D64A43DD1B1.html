TMMFDevSoundQueueItem Class Reference

class TMMFDevSoundQueueItem

TMMFDevSoundQueueItem is used to pass asynch messages between the devsound proxy and server for the observer's BufferToBeFilled, BufferToBeEmptied, PlayError, RecordError, ToneFinished, InitializeComplete and SendEventToClient events.
