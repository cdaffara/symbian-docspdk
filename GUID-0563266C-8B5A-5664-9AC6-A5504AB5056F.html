Porting Tutorial

Describes the steps to implement a port of the Digitizer Driver.

  • Set Up
    Describes how to use the template port to start your port.
  • Configuration: HAL (Hardware Abstraction Layer)
    HAL contains a number of attributes that are related to the Digitizer Driver.
  • Configuration: Timer Constants
    Describes how to set the values of timer constants that change the behaviour of the Digitizer Driver.
  • Channel Implementation
    Describes how to use the template port to implement the physical channel for the Digitizer Driver.
  • Calibration
    Calibration of the Digitizer Driver is required to convert between digitiser ADC values and screen coordinates.
