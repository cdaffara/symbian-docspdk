const TInt KHCIDataFramerInterfaceUid

const TIntKHCIDataFramerInterfaceUid

The UID of the HCI Data Framer API. If the HCI Data Framer API ever has to change, a new UID and associated M- class will be created. New implementations of Data Framer Plugin may implement the new API. Old (non-updated) HCI Data Framer Plugins will still work as long as the Stack supports the old API.
