Link Layer Utilities Overview

This topic describes the Link Layer Utilities collection.

Architecture

The Link Layer Utilities collection contains the Extensible Authentication Protocol (EAP) framework component. It works with the Point-to-Point Protocol to authenticate network connections.

Components

The EAP Framework provides support for EAP. EAP is a universal authentication framework for Ethernet type bearers.
