Feature Manager Querying Tutorial

This document introduces tutorials on the use of Feature Manager to query features.

Before you start, you must:

  • Understand what is meant by a feature, feature sets and feature data in Symbian platform.

  • Understand when to use CFeatureDiscovery or RFeatureControl to perform feature queries and handle feature notification.

A feature on Symbian platform encapsulates information such as the presence or absence of some item of hardware or software on the device. These tutorials explain how to use Feature Manager to query that information.
