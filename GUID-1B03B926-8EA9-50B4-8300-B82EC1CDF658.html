PSY Tester Reference

This section contains reference information for PSY Tester.

PSY Tester Menus Reference

The table below describes the purpose of the menu options of the PSY Tester menus.

Option Purpose

1-N: Select PSY

A user selects a PSY to test by entering a number between 1 and N from the keyboard, where N is the number of installed PSYs. Testing for the selected PSY starts when the joystick is pressed (Enter for the emulator). This choice is only available in the PSY Tester List View.

R: Test all PSYs

Tests all installed PSYs. Testing starts immediately after typing R. This choice is only available in the PSY Tester List View.

H: Help list

Displays help for the user. This choice is available in all views.

C: Cancel tests

Cancels any tests that are running. Tests are cancelled immediately and a cancellation message is shown to the user. This choice is only available in the PSY Tester List View.

U: Undo selection

Erases any numeric values typed in the List View or Settings View (for example a PSY index value). For example, this is used when the user wants to change the PSY to be tested.

L: List PSYs

Updates the List View to list the installed PSYs starting at index 1. The List View is updated immedately when L is pressed. This choice is only available in the PSY Tester List View.

Move joystick up / F / f: List former

Lists the PSYs immediately before those currently shown on the screen (if there are more PSYs than can be shown on a single screen). This choice is only available in the PSY Tester List view.

Move joystick down / N / n: List next

Lists the PSYs immediately after those currently shown on the screen (if there are more PSYs than can be shown on a single screen). This choice is only available in the PSY Tester List view.

Move joystick right: Shift view Joystick left: Shift view

Shifts between views. Views are arranged as follows:

Start View -> List View <-> Settings View.

It is nnot possible to move back to Start View having moved to List View.

X: Exit PSY Tester

Closes PSY Tester. Aborts any tests that are executing.
