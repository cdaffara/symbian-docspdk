Multimode Call Sub-session Tutorials

This section provides a summary of related tutorial of the multimode call API. The tutorials describe how to use different tasks in the multimode telephony.

  • Mobile Call Parameters Tutorial
    This tutorial describes how to set the mobile call parameters in the Multimode telephony API.
  • Call Information Tutorial
    This tutorial describes how to get the call information.
  • Call Status and Call Control Tutorial
    This tutorial describes how to manipulate the call status and control call information.
  • Emergency Call Tutorial
    This tutorial describes how to set up and make an emergency call.
  • Data Call Capability Tutorial
    This tutorial describes how to get the capabilities of a data call.
  • Data Call Parameters Tutorial
    This tutorial describes how to get the data call parameters from a call.
  • HSCSD Call Parameters Tutorial
    This tutorial explains how to get and set the High Speed Circuit Switched Data (HSCSD) call parameters.
  • Alternative Call Tutorial
    This tutorial describes how to use an alternative call.
  • CCBS Control Tutorial
    This tutorial describes how to control the Call Completion to Busy Subscriber (CCBS) service in a call.
  • User to User Signalling Service Tutorial
    This tutorial descrbes how to use the User to User Signalling (UUS) service in a call.
  • Multimedia Circuit Switched Call Settings Tutorial
    This tutorial describes how to setup and control a multimedia circuit switched (CS) call.
  • Traffic Channel Tutorial
    This tutorial describes how to set up a digital or analog traffic channel for a call in a CDMA network.
  • Voice Privacy Service Tutorial
    This tutorial describes how to use the voice privacy service in CDMA networks.
