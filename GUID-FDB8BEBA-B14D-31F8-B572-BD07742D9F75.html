LookupServicePresenceByServiceType(MServicePresenceInfo *&, const TDesC8 &)

voidLookupServicePresenceByServiceType(MServicePresenceInfo *&aServicePresence,
const TDesC8 &aMatch
)const [pure virtual]

Looks up a read-only service presence information object, having matching service type.

Parameters

MServicePresenceInfo *& aServicePresenceOn entry, contains the object from where to start the lookup or NULL if lookup should be started from the beginning of the set. If object given in entry is not found from set, lookup is started from the beginning of the set. On return, if an matching object is found, this is set to point to the found object, else NULL. Found object ownership is not returned to caller.
const TDesC8 & aMatchThe service type match pattern. Match pattern may contain question mark and asterisk characters ("?", "*") resepectively to match single or multiple characters. Service type are handled as folded in pattern matching.
