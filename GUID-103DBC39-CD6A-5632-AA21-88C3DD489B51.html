Properties Supported by File Data Provider

This topic lists the properties supported by the File Data Provider.

Properties Description

StorageID

Storage ID for this object.

ObjectFormat

MTP format code.

ProtectionStatus

Identifies the write-protection status of this object.

ObjectSize

Size of this object in bytes.

FileName

Indicates the file name of this object without specifying the directory or file structure details.

DateModified

Date on which this object was last modified.

ParentObject

Indicates the parent of this object.

PUOID

Persistent Unique Object Identifier.

Name

Contains the name of the object. In certain cases, this may overlap with other properties, such as FileName.

NonConsumable

Specifies whether the object is to be consumed by the device.
