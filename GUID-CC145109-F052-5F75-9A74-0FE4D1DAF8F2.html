Wireless LAN Parameters

Each WLAN parameter is stored in an Elementary File (EF), which is stored in a USIM Service Table (UST) on the Universal Subscriber Identity Module (USIM) of the phone. The ETelMM API allows the client to retrieve the following WLAN parameters:

  • Pseudonym

  • Re-authentication identity

  • User controlled list of WLAN PLMNs

  • Operator controlled list of WLAN PLMNs

  • User controlled list of WLAN SIDs (Specific Identifiers)

  • Operator controlled list of WLAN SIDs

Pseudonym

The Pseudonym is a temporary identifier used to find the subscriber identification. The Pseudonym is defined in EFPseudo. Service number 59 in the UST indicates that the USIM supports Pseudonym.

Re-authentication identity

The re-authentication identity is defined in EFWRI. This elementary file contains the following parameters that are used in the process of re-authentication: Master Key and Counter Value. Service number 66 in the UST indicates that USIM supports re-authentication.

User controlled list of WLAN PLMNs

The EFUPLMNWLAN contains the coding for preferred PLMNs and is used for WLAN PLMN selection. The information in this elementary file is determined by user. The first PLMN entry in the list indicates the highest priority and the last PLMN entry indicates the lowest. Service number 60 in the UST indicates that USIM supports a user controlled list of WLAN PLMNs.

Operator controlled list of WLAN PLMNs

The EFOPLMNWLAN contains the operator controlled list of PLMNs and is used for WLAN PLMN selection. The information in this elementary file is determined by the operator. The first PLMN entry in the list indicates the highest priority and the last PLMN entry indicates the lowest. Service number 61 in the UST indicates that USIM supports an operator controlled list of WLAN PLMNs.

User controlled list of WLAN SIDs

The EFUWSIDL contains the user controlled list of WLAN SIDs and is used for WLAN selection based on the priority defined by the user. The first entry in the list indicates the highest priority and the last entry indicates the lowest. Service number 62 in the UST indicates that USIM supports a user controlled list of WLAN SIDs.

Operator controlled list of WLAN SIDs

The EFOWSIDL contains the operator controlled list of WLAN SIDs and is used for WLAN selection based on the priority defined by the operator. The first entry in the list indicates the highest priority and the last entry indicates the lowest. Service number 63 in the UST indicates that USIM supports an operator controlled list of WLAN SIDs.
