E32ImportSection Class Reference

class E32ImportSection

Header for the Import Section in an image, as referenced by E32ImageHeader::iImportOffset. Immediately following this structure are an array of E32ImportBlock structures. The number of these is given by E32ImageHeader::iDllRefTableCount.
