TCatalogsLocalDebugHeapActivator Class Reference

class TCatalogsLocalDebugHeapActivator

Function-local debug heap activator. When instantiated as a local variable at start of a scope (e.g. function) makes sure debug heap is active only within the scope.

Note:

Cannot be used in scopes that leave item(s) on cleanup stack, such as an LC function.

Can be used in nested scopes.
