E32RelocSection Class Reference

class E32RelocSection

Header for a Relocation Section in an image, as referenced by E32ImageHeader::iCodeRelocOffset or E32ImageHeader::iDataRelocOffset.

Immediately following this structure are an array of E32RelocBlock structures.
