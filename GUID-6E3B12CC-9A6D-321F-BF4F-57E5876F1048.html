RSat::TGetInputV5 Class Reference

class RSat::TGetInputV5 : public RSat::TGetInputV1

Contains a GET INPUT proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifyGetInputPCmd() for more information about the command.

Request notification for GET INPUT with RSat::NotifyGetInputPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TGetInput" class. For Symbian OS versions up to and including v8.1a and 8.1b use RSat::TGetInputV1Pckg, a packaged RSat::TGetInputV1. For Symbian OS v9.2, use RSat::TGetInputV5. For later versions of Symbian OS, check for an updated API (RSat::TGetInputV6 etc).
