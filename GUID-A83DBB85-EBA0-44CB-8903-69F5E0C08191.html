Extended Connection Preference API- Tutorials

The Extended Connection Preference API allows the application to set preferences prior to connection establishment. These preferences enable and manage connections efficiently.

  • Opening a Connection to a SNAP by specifying SNAP Type
    Extended Connection Preferences API can be used by applications to set the SNAP type prior to establishing a connection.
  • Opening a Connection to a SNAP by specifying SNAP ID
    Extended Connection Preferences API can be used by applications to set the SNAP ID prior to establishing a connection.
  • Opening a Connection to an IAP
    Extended Connection Preferences API can be used by applications to set the preferred Internet Access Point (IAP) prior to establishing a connection.
  • Opening a Connection based on User Selection
    Extended Connection Preferences API can be used by applications to open a connection based on user selection.
  • Additional Connection Preferences
    Extended Connection Preferences API allows applications to set a few additional preferences that customize the connection establishment. These preferences, when used with specific SNAP ID, SNAP Type , IAP or connection selection dialog enable the applications to have more control on the connection.
