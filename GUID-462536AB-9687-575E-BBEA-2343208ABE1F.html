Core Telephony

The Core Telephony API provides a generic interface to start, control and terminate telephone calls. The core telephony functions are provided in terms of phone, line and call.

The core APIs provide the generic interface to the applications to initiate, control and terminate phone calls.

    The architecture section shows the relationship between the core telephony APIs and the functionality they provide to the client applications.

    The root session provides access to system telephony information, in particular the available phones and Telephony System Plug-ins (TSYs). The root session interface is provided by the RTelServer.
