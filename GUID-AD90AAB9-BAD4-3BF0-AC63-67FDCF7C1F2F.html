RSat::TSendSmV1 Class Reference

class RSat::TSendSmV1 : public RSat::TSatPCmdBase

Contains a SEND SHORT MESSAGE proactive command issued by the UICC. See ETSI TS 102 223 and RSat::NotifySendSmPCmd() for more information about the command.

Request notification for SEND SHORT MESSAGE with RSat::NotifySendSmPCmd(), passing it an object in which to store the command when it arrives. Pass a packaged version of the latest "TSendSm" class. For Symbian OS versions up to and including v8.1a and 8.1b use RSat::TSendSmV1Pckg, a packged RSat::TSendSmV1. For later versions of Symbian OS, check for an updated API (RSat::TSendSmV2 etc).
