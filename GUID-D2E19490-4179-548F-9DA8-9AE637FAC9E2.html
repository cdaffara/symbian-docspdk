Application Architecture Tutorials

This section describes the tasks that can be performed using the APIs provided by APPARC.

  • Application Service Tutorial
  • Accessing the Running Applications
    Applications running on a device are known as tasks. The TApaTaskList class is used to access applications running on a device, where each task can be an instance of the TApaTask class, which is used to manipulate or query tasks.
  • Porting Tutorial
  • Enumerating Applications
    Device creators can use the APIs provided by APPARC to enumerate a list of applications on a device.

  • Upgrading Help Files on the Secure Platform
    This page describes how to upgrade help files on the secure Symbian platform.
