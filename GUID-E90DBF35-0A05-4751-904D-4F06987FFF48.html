Memory Management

These documents describe how device drivers handle memory buffers and transfer data.

  • Data Buffers
    This document describes how device drivers allocate and access memory buffers.
  • Data Transfer across Memory Boundaries
    This document describes how to read from and write to user-side memory from a kernel thread context or a user thread.
  • Data Transfer between LDD and PDD
    This document describes how LDDs and PDDs exchange data.
  • Shared Chunks
    This document describes the use of shared chunks to share data.
