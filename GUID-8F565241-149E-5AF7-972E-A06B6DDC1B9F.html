Packet Data Telephony

This section provides the information related to the packet telephony API.

The Packet data API provides the interface to access the packet switched service. The packet data API functions allows to control, configure and get the network capabilities of the packet switched connection.

The packet telephony client is a PSD Agent that creates connections for an outbound interface like Point to Point Protocol (PPP) and Ethernet.
