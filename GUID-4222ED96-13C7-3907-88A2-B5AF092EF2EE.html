RTransferWindow Class Reference

class RTransferWindow

This class is a wrapper class to give a transfer buffer/window the same API as a CMMFDataBuffer. The purpose of this class is so that components such as codecs can use CMMFDataBuffers transparently without having to be concerned with whether the buffer is a descriptor buffer or a transfer buffer.

The class assumes that a valid RTransferWindow with a RTransferBuffer mapped in already exists.

Note: CMMFTransferBuffers are not supported in EKA2.
