const TInt KRemConBearerInterfaceUid

const TIntKRemConBearerInterfaceUid

The UID of the RemCon bearer interface. Implementors of new bearers are implementing this interface. The bearer interface is simply a means of supplying implementations of the 'bearer API' to RemCon. The bearer API may change over time, so this extension pattern is used.
