Network Request Handler

The Network Request Handler processes privacy and location requests received from the network.

  • Network Request Handler Overview
    The Network Request Handler (NRH) is responsible for processing and responding to privacy and location requests received from the network (MT-LR, Emergency MT-LR and NI-LR). It also provides an interface for installed applications to send the handsetâ€™s current location to remote third parties (MO-LR transmit to third party).
  • Privacy Requests
    A privacy request is a request for permission to obtain the mobile device's location. Privacy requests are usually sent from the network to the mobile device as part of an MT-LR.
