const TInt KHCTLInterfaceUid

const TIntKHCTLInterfaceUid

The UID of the hctl API. If the API ever has to change, a new UID and associated M class will be created. New implementations of CHCTLBase may implement the new API. Old (non-updated) implementations will still work as long as the old API is supported.
