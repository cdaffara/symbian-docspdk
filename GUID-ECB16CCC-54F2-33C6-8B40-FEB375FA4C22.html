const TInt KControllerInitialisationInterfaceUid

const TIntKControllerInitialisationInterfaceUid

The UID of the Controller Initialisation API. If the Controller Initialisation API ever has to change, a new UID and associated M- class will be created. New implementations of CControllerInitialisationPlugin may implement the new API. Old (non-updated) Controller Initialisation Plugins will still work as long as the stack supports the old API.
