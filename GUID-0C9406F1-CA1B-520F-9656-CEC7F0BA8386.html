Mailbox Dialling Numbers

Mailbox dialling number APIs are used to retrieve the phone numbers from the phone book. It is also used to retrieve pending messages. The mailbox dialling number feature already exists and the release 6 supports Video mail messages. Telephony Subsystem (TSY) supports mailbox dialling number functions.

The mailbox dialling numbers structures are represented in the following table:
