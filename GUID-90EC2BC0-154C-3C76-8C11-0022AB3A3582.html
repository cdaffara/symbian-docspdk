MSwtUiUtils Class Reference

class MSwtUiUtils : public MSwtAppFocusObserver

Interface for a utility class An instance (and only one) of the concrete class implementing this interface will be created by the Display when the UI thread is started. The instance will be destroyed when the UI thread is terminated.

SetSplitInputEditor(MSwtControl *)

voidSetSplitInputEditor(MSwtControl *aEditor)[pure virtual]

Rearranges the application layout (status pane, Shell position, split input view size) for best editing experience. There can be only one editor in split view mode at a time. Setting 0 clears the split view and restores the layout.

Following actions must be delegated to UiUtils while split editing is on:

  • Relocating the split input Shell (SetSplitInputShellPos)

  • Resizing the split input view. (SetSplitInputViewSize)Following actions must be notified to UiUtils while split editing is on:

  • Resizing the split input shell. (AdjustSplitInputShellPos)

  • Relocating the split input view. (AdjustSplitInputShellPos)

Parameters

MSwtControl * aEditor
