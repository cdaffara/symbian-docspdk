DirectGDI Component

The DirectGDI component provides a graphics context that can be hardware accelerated, and allows an asynchronous interface.

Target audience: Device creators.

Note: DirectGDI is deprecated in Symbian^3.

The CBR package name for the DirectGDI component is graphics_directgdi.
