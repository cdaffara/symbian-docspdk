USB Mass Storage File System

The USB client mass storage system is used to interact with media through USB when connected to a host (a PC or a Symbian platform device).

The USB host mass storage file system allows mass storage devices to connect to a Symbian platform device executing as the host.

  • USB Mass Storage Server
    Used to read and write data on a USB host device.
  • USB Host Mass Storage File System
    Provides an FSY plug-in for USB Mass Storage File System. Used to configure connection with attached devices so they can act as a mass storage device.
