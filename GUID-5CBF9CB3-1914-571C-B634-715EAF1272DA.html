Data Source Modules

The Data Source Modules collection contains components that enable the LBS subsystem to obtain position data.

  • Data Source Modules Overview
    The Data Source Modules collection contains components that enable the LBS subsystem to obtain position data.
  • Default Positioning Module
    The Default Positioning Module is a proxy PSY used by the Location Server (eposserver.exe) to select a PSY when a Location Acquisition API client opens a session without specifying a PSY ID.
  • Bluetooth GPS Positioning Module
    The Bluetooth GPS Positioning Module (Bluetooth GPS PSY) connects the LBS subsystem to an external Bluetooth GPS device.
  • Simulation Positioning Module
    The Simulation Positioning Module (Simulation PSY) enables application developers and device creators to test location-aware applications by providing location information without the need for a GPS module or a network connection.
  • GPS/A-GPS Positioning Module (PSY)
    The GPS/A-GPS PSY connects the Location Server to the A-GPS Location Manager.
  • Network Positioning Module
    The Network Positioning Module (PSY) connects the Location Server to the Network Location Manager.
