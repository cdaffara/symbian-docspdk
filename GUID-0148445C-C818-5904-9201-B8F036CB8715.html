Location Admin Overview

Location Admin is an LBS component that contains common LBS libraries grouped for convenience in a single system component.

Purpose

The LBS Admin component contains several unrelated APIs. Follow the links from the APIs section below to more information about the purpose of each of these APIs.

Required background

The reader should be familiar with the LBS architecture.

APIs

Several of the APIs contained by the Location Admin component are not directly related, but they may be broadly grouped by their intended audience.

APIs for application developers

Location Acquisition API

API Description

Location Acquisition API

This API is used by application developers to create location aware applications.

APIs for device creators

API Description

LBS Administration

This API is used to get and set the administration settings of the LBS subsystem.

A-GPS Location Data Source API

This API is used to create plug-ins that integrate with A-GPS hardware.

Network Protocol Module API

This API is used to create Network Protocol Modules.

LBS Logging API

This API is used to read LBS debug logs.

Clock Setting API

This API is used to set the system clock from GPS.
