TSoOwnerInfo Class Reference

class TSoOwnerInfo

Class used to communicate the owner of a socket from ESOCK to Provider.

Note that this only represents the thread/process that initially opened the socket and not, for example, any subsequent thread/process that issues IPC's on the socket from a shared session.
