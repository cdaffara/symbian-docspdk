CPlayer Class Reference

class CPlayer

The main responsibility of the Player is receiving the provider operations/queries from the Dealer and serving them using the protocol plug-ins. It is also responsible for adding/removing sub-sessions from the provider container in the related session object as well as keeping a list of all sub-sessions handled by the Player instance (iSubSessions).
