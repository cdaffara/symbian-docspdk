RMobileCall::TAudioToneV3 Class Reference

class RMobileCall::TAudioToneV3 : public RMobilePhone::TMultimodeType

Audio tone version 1 class. Contains information relating to the call control playing of audio tones. Tones maybe generated by the network (in-band) or signalled to be played locally by the ME.
