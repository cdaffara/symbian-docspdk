Privacy Protocol Module

The Privacy Protocol Module (PPM) is an optional LBS component that allows the LBS subsystem to process privacy requests from the device domestic OS. The privacy request may have been received from the network or from an installed application.

The Privacy Protocol Module supports two APIs that perform the same function: the Network Privacy API and the Privacy Request API.

  • Network Privacy API
    The Network Privacy API is an interface through which the LBS subsystem can receive and respond to privacy requests when it is built to include the Privacy Protocol Module.
