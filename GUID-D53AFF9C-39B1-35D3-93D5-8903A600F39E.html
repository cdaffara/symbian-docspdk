TRemConClientReceivePackage Struct Reference

struct TRemConClientReceivePackage

This struct wraps up the information passed from the server to the client in an RRemCon::Receive operation. It does not however include the arbitrary-length narrow descriptor 'associated data' as this cannot be wrapped in a struct. We do it this way because TIpcArgs only has four slots and we've used all of them.
