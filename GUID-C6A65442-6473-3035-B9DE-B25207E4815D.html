RSat::TPlayToneRspV1 Class Reference

class RSat::TPlayToneRspV1 : public RSat::TSatPCmdBase

Contains your response to a PLAY TONE proactive command. Pass this to the UICC with RSat::TerminalRsp(). See ETSI TS 102 223 and RSat::NotifyPlayTonePCmd() for more information about the command.

For Symbian OS versions up to and including v8.1a and v8.1b pass RSat::TerminalRsp() a RSat::TPlayToneRspV1Pckg (a packaged TPlayTonetRspV1). For later versions of Symbian OS, check for an updated API (RSat::TPlayToneRspV2 etc).
