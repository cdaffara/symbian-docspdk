RSat::TGetServiceInfoRspV2 Class Reference

class RSat::TGetServiceInfoRspV2 : public RSat::TSatPCmdBase

Contains your response to a GET SERVICE INFO proactive command. Pass this to the UICC with RSat::TerminalRsp(). See ETSI TS 102 223 and RSat::NotifyGetServiceInfoPCmd() for more information about the command.

For Symbian OS versions from 7.0s to v8.1a and 8.1b pass RSat::TerminalRsp() a RSat::TGetServiceInfoRspV2Pckg (a packaged TGetServiceInfoRspV2). For later versions of Symbian OS, check for an updated API (RSat::TGetServiceInfoRspV3 etc).
